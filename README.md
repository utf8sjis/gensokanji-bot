# gensokanji-bot

This is the [元素の漢字bot (@gensokanji_bot)](https://twitter.com/gensokanji_bot) a Twitter bot that regularly tweets the Chinese names of elements and trivia. This bot runs on GitHub Actions (scheduled workflows).
